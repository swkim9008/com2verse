Google Text to Speech
소개
구글 api를 활용한 text를 음성파일로 변환해주는 기능입니다.


기능
TTSManager
TTS 사용, 관리하는 클래스

TTSSecretary
외부에서 해당 패키지를 사용할때 접근하는 클래스

Secretary
실제 음성파일로 변환해주는 클래스
