# RenderTexture

## 소개

플랫폼별 지원하는 RenderTexture Format을 적용하고, 하면 해상도에 따라 RenderTexture의 해상도를 변경시킬 수 있는 기능 제작

<br>

## 기능

### VariableResolutionRenderTexture

일정량 이상 해상도 크기가 변화될 때 변경 비율에 맞게 렌더텍스쳐의 해상도를 변경해주는 클래스입니다.

### RenderTextureHelper

동적 렌더텍스쳐 생성을 도와주는 클래스입니다.
