# Com2Verse Web API
컴투버스 내 WebAPI 사용 편의를 위한 툴입니다.  
에디터 환경에서 테스트 웹 요청을 보내고, Swagger API를 파싱하여 API 호출을 랩핑하는 코드생성기를 제공합니다.
